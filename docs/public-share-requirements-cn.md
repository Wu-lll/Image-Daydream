# Image Daydream 公网分享版需求规格

## MVP 目标

朋友打开 Render 链接，输入访问口令，进入 Image Daydream，输入一句中文提示词，通常在 60 秒左右拿到一张图。

## 已锁定范围

- 整站访问口令，无账号系统。
- 36 小时未使用后重新输入口令。
- 默认显示 `线路 1 / 线路 2`，不暴露真实服务商名称。
- 站主 API Key 只放在 Render 环境变量中。
- 用户可以填写自己的 `Base URL + API Key + 模型名`。
- 用户可选择是否把自己的 Key 记住在当前浏览器。
- 保留尺寸、质量、格式、压缩、审核、数量、参考图等图片参数。
- 历史记录保存在浏览器本地，服务器不保存图片。
- 本地桥接保留为高级/开发模式，公网默认使用云端代理。

## 当前不做

- 不做注册登录。
- 不做数据库。
- 不做后台管理。
- 不做积分、计费、复杂额度系统。
- 不做自动线路切换。
- 不做图片云存储。
